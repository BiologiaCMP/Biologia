¿Qué es el metabolismo celular?

Es el conjunto de todas las reacciones químicas que ocurren dentro de las células para mantenerlas vivas y funcionando. Estas reacciones permiten obtener energía, construir estructuras, eliminar desechos y reproducirse.

Tipos de metabolismo

  • Anabolismo
  • Catabolismo

El anabolismo construye moléculas complejas a partir de otras simples, como construir proteínas a partir de aminoácidos. Necesita energía.

El catabolismo descompone moléculas complejas (como la glucosa) y libera energía, la cual se utiliza para actividades celulares.

Etapas principales del metabolismo

  • Glucólisis: rompe la glucosa en el citoplasma para obtener energía (ATP).
  • Ciclo de Krebs: ocurre en la mitocondria; produce CO₂ y más energía.
  • Cadena de transporte de electrones: genera la mayor parte del ATP.
