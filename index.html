¿Qué es el metabolismo celular?

Es el conjunto de reacciones químicas dentro de las células para obtener energía, construir estructuras y eliminar desechos.

Tipos de metabolismo

  • Anabolismo: construye moléculas complejas (consume energía).
  • Catabolismo: descompone moléculas para liberar energía.

¿Por qué es importante?

El metabolismo permite a las células mantenerse activas, crecer y realizar funciones vitales como la producción de ATP.
